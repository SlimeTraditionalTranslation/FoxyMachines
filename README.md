# Foxy 機器
Foxy 機器新增了各式各樣的機器與工具到 [Slimefun](https://github.com/Slimefun/Slimefun4) 之中。

> 此為**非官方**的黏液科技繁體翻譯版
>
> 請勿在該作者的問題追蹤中回報有關此的任何錯誤！
>
> 如果使用本翻譯遇到錯誤、翻譯用法奇怪或其他等等的問題
